8. Sort the list of words in descending order based on the number of occurrences.
9. Get the top-5 words with the most number of occurrences and return them.

Here's the Python code for the program:
```python
import os
from collections import Counter

def count_words(directory):
    word_counts = {}
    for filename in os.listdir(directory):
        with open(os.path.join(directory, filename), 'r') as file:
            file_words = file.read().split()
            word_counts.update(Counter(file_words))
    top_words = [word for word, count in word_counts.items() if count == word_counts.values().count(count)]
    return top_words[:5]
```

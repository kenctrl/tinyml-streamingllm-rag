To find the nth Fibonacci number using recursion, we can define a function that takes an integer n as input and returns the nth Fibonacci number. The first two Fibonacci numbers are 0 and 1, so we can start with those values and then compute the next numbers in the sequence.

```c++
#include <iostream>

using namespace std;

int fibonacci(int n) {
    if (n <= 1) {
        return n;
    }
    return fibonacci(n-1) + fibonacci(n-2);
}

    int n;

    return 0;
}
```
